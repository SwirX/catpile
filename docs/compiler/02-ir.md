# Compiler: Intermediate Representation (IR)

The IR is the bridge between parser and emitter. All parsers produce the same IR types, and the emitter consumes them uniformly.

<IRDiagram />

## IR Types

Located in `catpile/ir.py`:

### Program

Top-level - holds all scripts:

```python
@dataclass
class Program:
    scripts: list[ScriptDef]
```

### ScriptDef

One script (maps to the `script "name":` directive):

```python
@dataclass
class ScriptDef:
    name: str
    events: list[EventDef]
    functions: list[FunctionDef]
```

### EventDef

An event handler (maps to `on <event>: <body>`):

```python
@dataclass
class EventDef:
    kind: int          # Event ID (0=LOAED, 1=PRESSED, etc.)
    target: str | None  # UI element globalid or None
    body: list[Stmt]
```

### FunctionDef

A user-defined function:

```python
@dataclass
class FunctionDef:
    name: str
    params: list[str]
    body: list[Stmt]
```

### Statement Types

```python
@dataclass
class ActionStmt:
    name: str         # Action type string (e.g. "LOG", "VAR_SET")
    args: list[Arg]   # Typed arguments
    outputs: int = 0  # How many output variables

@dataclass
class IfStmt:
    condition: StrLit | VarRef  # IF condition
    body: list[Stmt]
    else_body: list[Stmt]

@dataclass
class RepeatStmt:
    times: str | None  # None = repeat_forever, string = count or {var}
    body: list[Stmt]

@dataclass
class ForEachStmt:
    table: str
    body: list[Stmt]

@dataclass
class BreakStmt:
    pass

@dataclass
class ReturnStmt:
    value: Arg | None
```

### Argument Types

```python
@dataclass
class StrLit:
    value: str

@dataclass
class VarRef:
    name: str          # Variable reference (l!name, or plain name)

@dataclass
class ObjectRef:
    name: str          # page.Element.Path reference (resolved by UI linker)

@dataclass
class NumLit:
    value: int | float

@dataclass
class InterpolatedStr:
    parts: list[StrLit | VarRef]  # Chain of string parts and var refs

@dataclass
class MathExpr:
    left: Arg
    op: str          # +, -, *, /, %
    right: Arg

@dataclass
class DictLiteral:
    entries: list[KVPair]  # Key-value pairs

@dataclass
class ListLiteral:
    elements: list[Arg]

@dataclass
class KVPair:
    key: str
    value: Arg

# Union type for all argument types
Arg = StrLit | VarRef | NumLit | InterpolatedStr | MathExpr | DictLiteral | ListLiteral | ObjectRef
Stmt = ActionStmt | IfStmt | RepeatStmt | ForEachStmt | BreakStmt | ReturnStmt
```

## IR Construction

The parser produces IR by walking the token stream. For example:

```python
# Source:
# if eq("{a}", "{b}"):
#     log("equal")

# Produces IR:
IfStmt(
    condition=StrLit("eq(\"{a}\", \"{b}\")"),
    body=[ActionStmt("LOG", [VarRef("a")])],
    else_body=[]
)
```

## IR Transformation

The optimizer transforms the IR by rewriting nodes:

```python
# Before: ActionStmt("LOG", [InterpolatedStr([StrLit("Hello "), VarRef("name")])])
# After optimization: ActionStmt("LOG", [MathExpr(...)])  if foldable
```

## IR Consumption

The emitter walks the IR recursively:

```python
def _emit_action(self, stmt: ActionStmt) -> list[dict]:
    return [M.make_action(stmt.name, *rendered_args, idgen=...)]
```
